****

## Run Locally


Clone the project

```bash
  git clone https://github.com/luisxsssx/server-local.git
```

Go to the project directory

```bash
  cd server-local
```

Access the frontend

``` bash
  cd client-files
```

Install dependencies in client-files

```bash
  npm install
```

Start the server

```bash
  npm run start
```

## On the backend

Access the backend

``` bash
  cd backend
```

Install dependencies in client-files

```bash
  mvn install
```

Start the server

```bash
  mvn spring-boot:run
```
